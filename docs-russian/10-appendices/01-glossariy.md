# ВЕЧНЫЙ ДВИГАТЕЛЬ
## Приложение A: Глоссарий терминов

---

## A

**ADX (Average Directional Index / Индекс среднего направленного движения)**  
Технический индикатор, разработанный Уэллсом Уайлдером для измерения силы тренда. Значения выше 25 указывают на сильный тренд; значения ниже 20 указывают на слабый тренд или боковое движение. Рассчитывается с использованием компонентов +DI и -DI.

**ATH (All-Time High / Исторический максимум)**  
Самая высокая цена, когда-либо достигнутая активом. Используется в Вечном Двигателе для расчёта просадок и активации тактического развёртывания.

**Альфа**  
Избыточная доходность относительно бенчмарка. В Вечном Двигателе альфа происходит от премии ребалансировки, захвата тренда и арбитража фандинга.

**Годовая доходность**  
Среднегодовая норма прибыли инвестиции за определённый период времени. Рассчитывается как: `(Конечная стоимость / Начальная стоимость)^(1/лет) - 1`

**Анти-Мартингейл**  
Подход к определению размера позиции, увеличивающий размер после выигрышей и уменьшающий после проигрышей. Вечный Двигатель использует инверсный Мартингейл (уменьшение размера во время просадок).

**Арбитраж**  
Одновременная покупка и продажа актива для получения прибыли от разницы в цене. В Вечном Двигателе арбитраж ставок фандинга использует разницу между спотовой ставкой и ставками фандинга перпетуалов.

**Распределение активов**  
Внедрение инвестиционной стратегии, пытающейся сбалансировать риск и доходность путём корректировки процентной доли каждого актива в инвестиционном портфеле.

**ATR (Average True Range / Средний истинный диапазон)**  
Индикатор волатильности, разработанный Уэллсом Уайлдером. Рассчитывается как 14-периодное скользящее среднее Истинного Диапазона. Используется для размещения стоп-лоссов и определения размера позиции.

---

## B

**Бэктест**  
Процесс тестирования торговой стратегии на исторических данных для оценки её эффективности. Вечный Двигатель требует 5+ лет бэктестов перед реальным развёртыванием.

**Базис**  
Разница между спотовой ценой и ценой фьючерса/перпетуала актива. В арбитраже фандинга риск базиса является основной проблемой.

**Медвежий рынок**  
Рыночное состояние, при котором цены падают на 20% или более от недавних максимумов на фоне широко распространённого пессимизма и негативного настроения инвесторов.

**Бета**  
Мера волатильности актива относительно общего рынка. Бета 1.0 означает, что актив движется вместе с рынком. Вечный Двигатель нацелен на стратегии с низкой бетой во время кризисов.

**Спред bid-ask**  
Разница между максимальной ценой, которую покупатель готов заплатить (bid), и минимальной ценой, которую продавец готов принять (ask). Важен для расчётов проскальзывания.

**Полосы Боллинджера**  
Инструмент технического анализа, определяемый набором трендовых линий, построенных на двух стандартных отклонениях от простого скользящего среднего (SMA).

**Бычий рынок**  
Финансовый рынок группы ценных бумаг, на котором цены растут или ожидается их рост. Обычно определяется как рост на 20% от недавних минимумов.

---

## C

**CAGR (Compound Annual Growth Rate / Среднегодовой темп роста)**  
Среднегодовой темп роста инвестиции за определённый период времени более одного года. Формула: `(Конечная стоимость / Начальная стоимость)^(1/n) - 1`

**Календарная ребалансировка**  
Ребалансировка портфеля через заранее определённые интервалы времени (например, ежемесячно, ежеквартально). Вечный Двигатель использует ежеквартальную календарную ребалансировку с пороговыми триггерами.

**Коэффициент Калмара**  
Сравнение среднегодовой скорректированной инфляцией нормы доходности и риска максимальной просадки. Рассчитывается как: `CAGR / Макс. просадка`

**Коэффициент CAPE (Cyclically Adjusted PE / Циклически скорректированное P/E)**  
Мера оценки, использующая реальную прибыль на акцию за 10-летний период. Разработан Робертом Шиллером. Используется для долгосрочной оценки рынка.

**Сохранение капитала**  
Инвестиционная стратегия, которая отдаёт приоритет предотвращению убытков перед достижением прибыли. Первый принцип Вечного Двигателя.

**Автостоп**  
Автоматический торговый останов, предназначенный для сдерживания панических продаж на фондовых биржах. Вечный Двигатель реализует четыре уровня автостопов при просадках 10%, 15%, 20% и 25%.

**Сложные проценты**  
Проценты, начисляемые на начальную сумму и на накопленные проценты предыдущих периодов. «Проценты на проценты».

**CVaR (Conditional Value at Risk / Условное значение риска)**  
Также известно как Expected Shortfall (Ожидаемый дефицит) — среднее из худших потерь за пределами порога VaR.

**Корреляция**  
Статистика, измеряющая степень, в которой две ценные бумаги движутся относительно друг друга. Диапазон от -1 (идеальная отрицательная) до +1 (идеальная положительная).

**Матрица корреляций**  
Таблица, показывающая коэффициенты корреляции между переменными. Используется в Вечном Двигателе для мониторинга разрушения диверсификации.

**Контрагентский риск**  
Риск того, что другая сторона в инвестиционной, кредитной или торговой сделке может не выполнить свою часть сделки и может не выполнить договорные обязательства.

**Кросс-коллатераль**  
Режим маржи, при котором несколько активов могут использоваться в качестве залога для торговых позиций. Функция Bybit UTA.

---

## D

**DCA (Dollar-Cost Averaging / Усреднение долларовой стоимости)**  
Инвестиционная стратегия, при которой инвестор распределяет общую сумму инвестиций на периодические покупки целевого актива для снижения влияния волатильности.

**Дельта**  
Соотношение, сравнивающее изменение цены актива с соответствующим изменением цены его производной. Дельта-нейтраль означает нулевую чувствительность к движениям цен.

**Дельта-нейтраль**  
Стратегия портфеля, использующая несколько позиций с балансировкой положительных и отрицательных дельт, так что общая дельта активов в целом равна нулю.

**Индикатор Демарка**  
Инструмент технического анализа, который сравнивает максимальные и минимальные цены последнего периода с эквивалентными ценами предыдущего периода.

**Деривативы**  
Финансовые ценные бумаги, стоимость которых зависит от или производна от базового актива или группы активов. Включает фьючерсы, опционы и перпетуалы.

**Канал Дончиана**  
Технический индикатор, разработанный Ричардом Дончианом. Формирует полосу с верхней полосой = максимуму за N периодов, нижней полосой = минимуму за N периодов.

**Просадка**  
Снижение от пика до минимума в течение определённого периода инвестиций, фонда или товара. Просадка 20% означает снижение на 20% от пика.

**Двойной импульс**  
Инвестиционная стратегия, сочетающая импульс относительной силы (выбор актива) и абсолютный импульс (следование тренду). Разработана Гэри Антоначчи.

---

## E

**EMA (Exponential Moving Average / Экспоненциальное скользящее среднее)**  
Тип скользящего среднего, который придаёт больший вес и значимость последним точкам данных. Вечный Двигатель использует EMA 50 и 200 периодов.

**Кривая капитала**  
Графическое представление изменения стоимости торгового счёта за период времени.

**EV (Expected Value / Ожидаемое значение)**  
Ожидаемая ценность данной инвестиции. Рассчитывается как: `Σ(Вероятность × Результат)`

**Exchange Broker**  
Функция на Bybit, позволяющая управлять несколькими субаккаунтами с пользовательскими лимитами скорости и разрешениями.

---

## F

**Толстый хвост**  
Статистическое явление, при котором вероятность экстремальных исходов выше, чем предсказывается нормальным распределением. Распространено на криптовалютных рынках.

**Фиат**  
Выпущенная правительством валюта, которая не обеспечена физическим товаром, таким как золото или серебро, а поддерживается правительством, его выпустившим.

**Фиксированная дробная позиция**  
Техника управления капиталом, при которой размер позиции основан на фиксированном проценте от капитала счёта. Вечный Двигатель использует риск 1% на сделку.

**FOK (Fill or Kill / Исполнить или отменить)**  
Тип ордера, который должен быть исполнен немедленно полностью или не исполнен вовсе.

**FOMO (Fear of Missing Out / Страх упустить выгоду)**  
Поведенческое явление, при котором инвесторы покупают активы из-за страха пропустить потенциальные прибыли. Вечный Двигатель устраняет FOMO через систематическое исполнение.

**Ставка фандинга**  
Периодические платежи, производимые трейдерам или ими на основе разницы между ценой перпетуала и спотовой ценой. Выплачиваются каждые 8 часов на Bybit.

**Фьючерсы**  
Юридическое соглашение о покупке или продаже определённого товара или актива по заранее определённой цене в указанное время в будущем.

---

## G

**GTC (Good Till Canceled / Действует до отмены)**  
Ордер на покупку или продажу ценной бумаги, который остаётся в силе до тех пор, пока ордер не будет исполнен или инвестор не отменит его.

---

## H

**HODL**  
Термин, произошедший от опечатки слова «hold», который относится к стратегиям buy-and-hold в криптовалюте. Основная философия движка CORE-HODL.

---

## I

**Iceberg Order / Айсберг-ордер**  
Большой одиночный ордер, разделённый на более мелкие лимитные ордера, обычно с помощью автоматизированной программы, для скрытия фактического количества ордера.

**IMR (Initial Margin Rate / Начальная маржинальная ставка)**  
Минимальное количество капитала, необходимое для открытия позиции с плечом. На Bybit 100% IMR означает, что новые позиции не могут быть открыты.

**Индексная цена**  
Совокупная цена актива, полученная из нескольких бирж. Используется для расчёта цены маркировки и ставок фандинга.

**IOC (Immediate or Cancel / Немедленно или отменить)**  
Ордер на покупку или продажу, который должен быть исполнен немедленно. Любая часть ордера, которая не может быть заполнена немедленно, отменяется.

---

## K

**Критерий Келли**  
Математическая формула для определения размера ставки: `f = (bp - q) / b`, где f — доля капитала, b — шансы, p — вероятность выигрыша, q — вероятность проигрыша. Вечный Двигатель использует 1/8 Келли для безопасности.

**Аварийный выключатель**  
Механизм экстренного отключения, который немедленно останавливает всю торговую активность. Активируется при Автостопе Уровень 4.

---

## L

**Задержка**  
Временная задержка между инициированием запроса и ответом. Критична для высокочастотных стратегий; менее критична для более длинных таймфреймов Вечного Двигателя.

**Layer 1 (Блокчейн)**  
Базовая сеть или сам блокчейн (например, Bitcoin, Ethereum). CORE-HODL Вечного Двигателя фокусируется на крупных активах Layer 1.

**Плечо**  
Использование заёмного капитала для увеличения потенциальной доходности инвестиции. Вечный Двигатель ограничивает плечо максимум 2x.

**Лимитный ордер**  
Ордер на покупку или продажу ценной бумаги по определённой цене или лучше.

**Ликвидность**  
Степень, в которой актив может быть быстро куплен или продан без влияния на его цену. Существенна для входа и выхода из позиций.

---

## M

**Комиссия мейкера**  
Комиссия, уплачиваемая трейдерами, которые добавляют ликвидность к стакану, размещая лимитные ордера. Ниже, чем комиссия тейкера на Bybit (0.01% против 0.06%).

**Цена маркировки**  
Цена, используемая для расчёта нереализованного P&L и триггеров ликвидации. Получена из индексной цены для предотвращения манипуляций.

**Рыночный ордер**  
Ордер на немедленную покупку или продажу по лучшей доступной текущей цене.

**Рыночный режим**  
Текущее состояние рынка (тренд вверх, тренд вниз, боковик, волатильность). Вечный Двигатель определяет режимы для корректировки распределений.

**Максимальная просадка (MDD)**  
Максимальная зафиксированная потеря от пика до минимума портфеля, прежде чем будет достигнут новый пик. Вечный Двигатель ограничивает MDD до 35%.

**MMR (Maintenance Margin Rate / Ставка поддерживающей маржи)**  
Минимальный капитал, необходимый для поддержания открытой позиции. Когда MMR достигает 100%, происходит ликвидация.

**Моделирование Монте-Карло**  
Метод оценки вероятности различных исходов путём проведения множества испытаний с использованием случайных переменных. Используется для валидации прогнозов.

**Скользящее среднее**  
Расчёт для анализа точек данных путём создания серии средних значений различных подмножеств полного набора данных.

**Мульти-стратегия**  
Подход, сочетающий несколько торговых стратегий для достижения лучших риск-скорректированных доходностей. Вечный Двигатель использует четыре различные стратегии.

---

## O

**Открытый интерес**  
Общее количество непогашённых деривативных контрактов, таких как опционы или фьючерсы, которые не были урегулированы.

**Стакан ордеров**  
Электронный список ордеров на покупку и продажу определённой ценной бумаги или финансового инструмента, организованный по уровням цен.

---

## P

**Перпетуальные фьючерсы**  
Тип фьючерсного контракта без даты истечения. Используют ставки фандинга для привязки цены к споту. Основной инструмент для TREND движка Вечного Двигателя.

**Определение размера позиции**  
Процесс определения объёма капитала для выделения на конкретную сделку. Критично для управления риском.

**Post-Only**  
Тип ордера, который гарантирует, что ордер добавляет ликвидность к стакану (мейкер) и не исполняется немедленно против существующих ордеров (тейкер).

**Профит-фактор**  
Торговая метрика, рассчитываемая как: `Валовая прибыль / Валовый убыток`. Значения выше 1.5 указывают на прибыльные стратегии.

---

## R

**R-множитель**  
Способ выразить прибыль или убыток как множитель от начального риска (R). 3R победитель принёс 3 раза больше суммы риска.

**Лимит скорости**  
Максимальное количество API-запросов, разрешённых в период времени. Bybit позволяет 120 запросов в минуту по умолчанию.

**Ребалансировка**  
Процесс перераспределения весов портфеля активов путём покупки или продажи активов для поддержания желаемого распределения активов.

**Обнаружение режима**  
Определение текущего состояния рынка (тренд, боковик, волатильность) для корректировки параметров стратегии.

**REST API**  
Representational State Transfer Application Programming Interface. Стандартный способ взаимодействия систем через HTTP.

**Риск-паритет**  
Подход к управлению инвестициями, который фокусируется на распределении риска, а не распределении капитала.

**Риск-скорректированная доходность**  
Расчёт доходности от инвестиции, который учитывает степень риска, которую необходимо принять для достижения этой доходности. Измеряется коэффициентом Шарпа.

**ROI (Return on Investment / Возврат на инвестиции)**  
Мера производительности, используемая для оценки эффективности инвестиции. Рассчитывается как: `(Текущая стоимость - Затраты) / Затраты`

**RSI (Relative Strength Index / Индекс относительной силы)**  
Осциллятор импульса, измеряющий скорость и изменение движений цен. Диапазон 0-100; обычно 70 = перекупленность, 30 = перепроданность.

---

## S

**Коэффициент Шарпа**  
Мера риск-скорректированной доходности. Рассчитывается как: `(Доходность - Безрисковая ставка) / Стандартное отклонение`. Вечный Двигатель нацеливается на Шарпа > 1.3.

**Проскальзывание**  
Разница между ожидаемой ценой сделки и фактической ценой, по которой исполняется сделка. Цель <0.3%.

**SMA (Simple Moving Average / Простое скользящее среднее)**  
Арифметическое скользящее среднее, рассчитываемое путём сложения последних цен и последующего деления этой цифры на количество временных периодов в расчётном среднем.

**Коэффициент Сортино**  
Вариация коэффициента Шарпа, которая различает вредную волатильность от общей волатильности. Использует только нисходящее отклонение.

**Спотовый рынок**  
Рынок, на котором финансовые инструменты или товары торгуются для немедленной доставки.

**Стейблкоин**  
Тип криптовалюты, разработанный для поддержания стабильной стоимости относительно фиатной валюты (обычно USD). USDT и USDC — примеры.

**Стандартное отклонение**  
Мера величины вариации или разброса набора значений. Используется для количественной оценки волатильности.

**Стоп-лосс**  
Ордер, размещённый у брокера на покупку или продажу после достижения акцией определённой цены. Предназначен для ограничения убытков инвестора.

**Деградация стратегии**  
Явление, при котором производительность торговой стратегии ухудшается со временем по мере изменения рыночных условий или увеличения числа участников, использующих то же преимущество.

**Субаккаунт**  
Отдельный аккаунт под мастер-аккаунтом, используемый для изоляции риска и организации торговой деятельности.

---

## T

**Комиссия тейкера**  
Комиссия, уплачиваемая трейдерами, которые удаляют ликвидность из стакана, исполняясь против существующих ордеров. Выше, чем комиссии мейкера.

**Тактическое распределение активов**  
Стратегия управления портфелем активного управления, которая смещает процент активов, удерживаемых в различных категориях, для использования аномалий рыночного ценообразования или сильных рыночных секторов.

**Технический анализ**  
Дисциплина торговли, используемая для оценки инвестиций и выявления торговых возможностей путём анализа статистических трендов, полученных из торговой активности.

**Пороговая ребалансировка**  
Ребалансировка, активируемая когда распределение актива отклоняется за предварительно определённый процент от целевого распределения.

**TWAP (Time-Weighted Average Price / Средневзвешенная по времени цена)**  
Средняя цена ценной бумаги за указанный период времени. Используется для исполнения крупных ордеров.

**Следование тренду**  
Торговая стратегия, пытающаяся получить прибыль через анализ импульса актива в определённом направлении.

**Истинный диапазон**  
Наибольшее из: (текущий high - текущий low), |текущий high - предыдущее close|, |текущий low - предыдущее close|. Используется в расчёте ATR.

---

## U

**UTA (Unified Trading Account / Единый торговый счёт)**  
Система счетов Bybit, позволяющая торговать спот, деривативы и опционы из одного счёта с общей маржей.

---

## V

**VaR (Value at Risk / Значение риска)**  
Статистика, которая количественно оценивает степень возможных финансовых потерь внутри фирмы, портфеля или позиции за конкретный период времени.

**Волатильность**  
Статистическая мера разброса доходностей данной ценной бумаги или рыночного индекса. Обычно измеряется стандартным отклонением.

**Таргетинг волатильности**  
Корректировка размеров позиций на основе текущей рыночной волатильности для поддержания постоянного портфельного риска.

---

## W

**WebSocket**  
Протокол компьютерных коммуникаций, обеспечивающий полнодуплексные каналы связи через одно TCP-соединение. Используется для рыночных данных в реальном времени.

**Взбивание**  
Состояние, при котором цена ценной бумаги движется в одном направлении, но затем быстро следует движение в противоположном направлении.

**Винрейт**  
Процент сделок, которые являются прибыльными. Следование тренду Вечного Двигателя нацелено на винрейт 35-40% с высокими соотношениями риск/прибыль.

---

## Y

**Доходность**  
Доход от инвестиции. В Вечном Двигателе доходность поступает от ставок фандинга, стейкинга и Bybit Earn.

# СПРАВОЧНИК ФОРМУЛ

## Определение размера позиции

**Критерий Келли (Полный):**
```
f = (bp - q) / b

f = доля капитала для риска
b = средний выигрыш / средний проигрыш (шансы)
p = вероятность выигрыша
q = вероятность проигрыша (1 - p)
```

**Дробный Келли (Вечный Двигатель):**
```
Размер позиции = (Счёт × Доля Келли × Келли) / Расстояние стопа

Доля Келли = 0.125 (1/8)
Макс. риск = 1% от счёта на сделку
```

## Метрики риска

**Коэффициент Шарпа:**
```
Шарп = (Rp - Rf) / σp

Rp = Доходность портфеля
Rf = Безрисковая ставка
σp = Стандартное отклонение доходности портфеля
```

**Максимальная просадка:**
```
MDD = (Минимальная стоимость - Максимальная стоимость) / Максимальная стоимость
```

**VaR (Параметрический):**
```
VaR = Стоимость портфеля × (μ - zσ)

μ = Ожидаемая доходность
z = Z-score для уровня доверия (1.65 для 95%)
σ = Стандартное отклонение
```

**Коэффициент Калмара:**
```
Калмар = CAGR / Макс. просадка
```

## Технические индикаторы

**EMA (Экспоненциальное скользящее среднее):**
```
EMA_t = (Close_t × k) + (EMA_{t-1} × (1 - k))

k = 2 / (N + 1)
N = Период
```

**ATR (Средний истинный диапазон):**
```
TR = max[(High - Low), |High - Close_{t-1}|, |Low - Close_{t-1}|]
ATR = SMA(TR, 14)
```

**ADX (Индекс среднего направленного движения):**
```
+DM = Текущий High - Предыдущий High (если положительный, иначе 0)
-DM = Предыдущий Low - Текущий Low (если положительный, иначе 0)
+DI = 100 × EMA(+DM, 14) / ATR
-DI = 100 × EMA(-DM, 14) / ATR
DX = 100 × |+DI - -DI| / (+DI + -DI)
ADX = EMA(DX, 14)
```

## Математика портфеля

**Среднегодовой темп роста:**
```
CAGR = (Конечная стоимость / Начальная стоимость)^(1/лет) - 1
```

**Доходность портфеля с взносами:**
```
FV = P(1 + r)^n + PMT × [((1 + r)^n - 1) / r]

P = Основная сумма
PMT = Периодический взнос
r = Периодическая норма доходности
n = Количество периодов
```

**Корреляция:**
```
ρ(X,Y) = Cov(X,Y) / (σX × σY)
```
