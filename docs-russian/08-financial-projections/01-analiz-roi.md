# ЧАСТЬ I: ФЕНОМЕН СЛОЖНЫХ ПРОЦЕНТОВ

## Глава 1: Математика богатства

### 1.1 Сила экспоненциального роста

Альберт Эйнштейн, как сообщается, называл сложные проценты «восьмым чудом света». На рынках криптовалют эта сила усиливается за счёт более высокой базовой доходности.

**Формула сложных процентов:**
```
A = P × (1 + r)^n

Где:
  A = Итоговая сумма
  P = Начальный капитал
  r = Годовая норма доходности
  n = Количество лет
```

**Пример:** $10,000 при 20% на 20 лет
```
A = $10,000 × (1 + 0.20)^20
A = $10,000 × 38.34
A = $383,400
```

Это **возврат в 38 раз** от начальных инвестиций.

### 1.2 Влияние регулярных взносов

Добавление ежемесячных взносов драматически ускоряет создание богатства:

```
FV = P × (1 + r)^n + PMT × [((1 + r)^n - 1) / r]

Где:
  PMT = Ежемесячный взнос
  r = Месячная норма доходности (годовая / 12)
  n = Количество месяцев
```

**Пример:** $10,000 начальных + $500/месяц при 20% на 20 лет
```
FV = $383,400 (от начального) + $5,800,000 (от взносов)
FV = $6,183,400
```

**97% итоговой стоимости приходится на сложные проценты, а не на взносы.**

### 1.3 Правило 72 (и его вариации)

Быстрый расчёт времени удвоения:

| Годовая доходность | Лет до удвоения | Лет до 10-кратного роста |
|-------------------|-----------------|--------------------------|
| 10% (S&P 500) | 7.2 года | 24 года |
| 15% (Консервативный) | 4.8 года | 16 лет |
| 20% (Базовый сценарий) | 3.6 года | 12 лет |
| 30% (Оптимистичный) | 2.4 года | 8 лет |

При доходности 20% ваши деньги удваиваются каждые 3.6 года.

### 6.1 Понимание коэффициента Шарпа

Коэффициент Шарпа измеряет доходность на единицу риска:

```
Коэффициент Шарпа = (Доходность - Безрисковая ставка) / Стандартное отклонение
```

| Хедж-фонды (средние) | 8% | 10% | 0.60 |

### 6.2 Коэффициент Сортино (только нисходящий риск)

```
Коэффициент Сортино = (Доходность - Безрисковая ставка) / Нисходящее отклонение
```

# ЗАКЛЮЧЕНИЕ: УРАВНЕНИЕ БОГАТСТВА

Вечный Двигатель представляет математическую определённость:

**Сложный рост + Время - Катастрофические потери = Межпоколенческое богатство**

**Ключевые выводы:**

1. **Даже консервативные сценарии** (12%) дают 7-кратный возврат за 20 лет
2. **Базовые сценарии** (20%) дают 33-кратный возврат — богатство, меняющее жизнь
3. **Просадки временны** — восстановление математически определено при наличии времени
4. **Затраты минимальны** — автоматизация устраняет комиссионные издержки
5. **Налоговая эффективность имеет значение** — наш подход экономит миллионы по сравнению с альтернативами

**Формула вашего богатства:**

```
Итоговая стоимость = Начальные × (1 + Доходность)^Годы + 
                     Ежемесячный × [((1 + Доходность)^Годы - 1) / Доходность]

Где:
  Начальные = С чего вы начинаете
  Доходность = 15-25% (от консервативного до базового)
  Годы = Ваш временной горизонт (10, 20, 30)
  Ежемесячный = Постоянные взносы
```

**Единственная переменная, которую вы контролируете:**  
Время. Начните сейчас.
